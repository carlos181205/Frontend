## ⚡ Resumen rápido

- **Actividad 1:** API REST (backend) con CRUD completo de `orders` + gestión de items + productos + health + docs.
- **Actividad 2:** Frontend en Next.js + Tailwind CSS que consuma la API de la Actividad 1.
- **Entrega mínima:** Todos los endpoints marcados como **Obligatorios** deben funcionar.
- **Sin datos hardcodeados** en el frontend — debe consumir la API real desplegada.
